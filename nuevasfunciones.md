# Roadmap de Nuevas Funciones para Peluquería SaaS

A continuación se detallan las funcionalidades recomendadas para elevar el sistema a un nivel competitivo de mercado.

## 1. Notificaciones y Comunicación
- [ ] **Notificaciones por WhatsApp:** Envío automático de recordatorios de turno 2 horas antes de la cita.
- [ ] **Emails Transaccionales:** Confirmación de reserva, cancelación y bienvenida con diseño premium.
- [ ] **Alertas de Push:** Para usuarios con la app abierta o instalada como PWA.

## 2. Gestión de Negocio Avanzada
- [ ] **Gestión de Empleados:** Dashboard completo para que cada peluquero vea su propia agenda.
- [ ] **Comisiones:** Cálculo automático de comisiones por servicio para cada empleado.
- [ ] **Reportes y Analytics:** Gráficos de recaudación mensual, servicios más pedidos y horas pico.
- [ ] **Control de Stock:** Registro de productos (tinturas, shampoos) utilizados en cada servicio.

## 3. Experiencia del Cliente (CRM)
- [ ] **Ficha de Cliente:** Notas sobre preferencias (ej. "usa tintura nro 5", "prefiere café sin azúcar").
- [ ] **Historial de Fotos:** Galería de fotos de los cortes/colores anteriores del cliente.
- [ ] **Sistema de Puntos/Lealtad:** Descuentos automáticos tras X cantidad de visitas.
- [ ] **Valoraciones y Reseñas:** Feedback de clientes post-servicio visible en la página pública.

## 4. Pagos y Finanzas
- [ ] **Integración Real de Pagos:** Conexión con Mercado Pago o Stripe para cobros de señas o abonos.
- [ ] **Facturación Automática:** Generación de recibos digitales.
- [ ] **Gestión de Gastos:** Registro de costos fijos (luz, alquiler) para ver rentabilidad neta.

## 5. Personalización y Marca
- [ ] **Branding Personalizado:** Permitir que cada peluquería suba su logo y elija sus colores principales.
- [ ] **Galería de Portfolio:** Carrusel de fotos de trabajos realizados en la página de reserva pública.
- [ ] **Múltiples Sucursales:** Gestión centralizada para dueños de varias peluquerías.

## 6. Optimizaciones Técnicas
- [ ] **PWA (Progressive Web App):** Para que los clientes puedan "instalar" el sistema en su celular.
- [ ] **Optimización de Imágenes:** Procesamiento automático de fotos para carga ultra rápida.
- [ ] **Soporte Multidioma:** Preparar el sistema para otros mercados (Inglés/Portugués).
